# NgBootstrap Snippets

A set of NgBootstrap (Angular 2 + Bootstrap 4) Snippets for Visual Studio Code.

Create a new HTML document and type 'ngb' to see all available snippets.

## Available Snippets

### Alerts

Trigger | Description
--- | ---
ngb-alert | Alert

### Buttons
Trigger | Description
--- | ---
ngb-button | Button
